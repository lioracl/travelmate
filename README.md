# TravelMate

האתר פוצל לקבצים נפרדים כדי שיהיה קל לערוך ולתחזק אותו:

- `index.html` — מסך רשימת הטיולים
- `trip/japan-2027/index.html` — עמוד הטיול ליפן
- `trip/italy-2028/index.html` — עמוד הטיול לאיטליה
- `assets/styles.css` — כל העיצוב המשותף
- `assets/app.js` — כל ההתנהגות והאינטראקציות

## תצוגה מקומית

יש להפעיל שרת מקומי מתיקיית הפרויקט, משום שהקישורים באתר משתמשים בנתיבים שמתחילים ב־`/`.

לדוגמה, אם Python מותקן:

```powershell
python -m http.server 8000
```

לאחר מכן פותחים בדפדפן: `http://localhost:8000`

האתר עדיין טוען גופן, אייקונים ותמונות ממקורות חיצוניים, ולכן נדרש חיבור לאינטרנט להצגה זהה למקור.
