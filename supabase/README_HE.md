# הפעלת כספת המסמכים של TravelMate

הקוד באתר משתמש רק ב־Publishable key. אין להכניס לקוד מפתח `service_role`, מפתח סודי או סיסמת מסד נתונים.

## 1. יצירת הטבלאות והאחסון הפרטי

1. פתחו את פרויקט `travelmate` ב־Supabase.
2. בתפריט הצד בחרו **SQL Editor**.
3. לחצו **New query**.
4. העתיקו את כל תוכן הקובץ `travelmate.sql` והדביקו בעורך.
5. לחצו **Run** פעם אחת.

הסקריפט יוצר:

- טבלת `travel_documents` עם Row Level Security.
- Bucket פרטי בשם `travel-documents`.
- הרשאות שמגבילות כל משתמש לתיקייה הנושאת את מזהה החשבון שלו.
- מגבלת קובץ של 25MB ואפשרות לשמור רק קבצים מוצפנים.

## 2. כתובת החזרה לאחר אימות דוא״ל

1. עברו אל **Authentication → URL Configuration**.
2. הגדירו את **Site URL** בתור:

   `https://lioracl.github.io/travelmate/`

3. הוסיפו אל **Redirect URLs**:

   `https://lioracl.github.io/travelmate/**`

## 3. בדיקה

1. פתחו טיול מותאם אישית ועברו למסמכים.
2. צרו חשבון באמצעות דוא״ל וסיסמה.
3. אשרו את הדוא״ל אם התקבל מייל אימות.
4. התחברו, בחרו סיסמת הצפנת כספת והעלו מסמך בדיקה שאינו רגיש.
5. פתחו את אותו טיול במכשיר אחר, התחברו והזינו את אותה סיסמת הצפנה.

סיסמת הצפנת הכספת אינה נשלחת ל־Supabase ואינה נשמרת בדפדפן. אובדן הסיסמה ימנע שחזור של הקבצים המוצפנים.
